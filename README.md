# LIKELION Backend Development Journey (Spring) 🦁

## 📌 소개
멋쟁이사자처럼 활동을 통해  
Spring 기반 백엔드 개발을 학습하며 성장 과정을 기록한 저장소입니다.  

또한 별도의 스터디를 병행하며  
학습한 내용을 정리하고, 직접 구현을 통해 이해도를 높이고 있습니다.

---

## 🎯 목표
- Spring 기반 백엔드 개발 역량 강화
- 서버 구조 및 데이터 흐름 이해
- 실무형 프로젝트 경험 축적

---

- Java 기반 서버 개발
- Spring 프레임워크 학습
- REST API 설계 및 구현

### 🔹 데이터 처리
- 데이터 흐름 처리 및 구조 설계
- (추후) DB 연동 및 관리

### 🔹 스터디 병행
- 별도의 스터디를 통해 학습 내용 정리
- 코드 구현 및 복습 중심 학습
- 문제 해결 과정 공유 및 개선

### 🔹 협업 및 버전 관리
- Git / GitHub 활용
- 협업 및 코드 관리 경험
